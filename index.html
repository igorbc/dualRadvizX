Project overview

This application is intended to be used to visualize both classification results and data attributes. By using the slider you can switch between each of this views in a continuous way. For now there's only one set of data to be explored, which consists of 50 instances from the Iris dataset classified using Na√Øve Bayes (the remaining instances were used for training the classifier).
